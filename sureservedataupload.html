📋 CSV Format Requirements
Your CSV file should contain the following columns:
  • Column B (index 1): Ref (Job Reference)
  • Column C (index 2): Description (Job Type)
  • Column D (index 3): Date of Appointment
  • Column E (index 4): Points Value ← read directly from CSV
  • Column G (index 6): Resource (Engineer)
  • Column R (index 17): Status (Job Outcome)

Points values are read directly from Column E and stored in Firebase. Jobs with no value in Column E will be flagged.
